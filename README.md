Pull in the Sources
======================
Create Folder for Repo
-----------------------
$ mkdir Unicornblood_N && cd Unicornblood_N

Sync in Source
---------------
$ repo init -u git://github.com/NurKeinKernel/Unicornblood -b n && repo sync

Build Command
-----------------
$ ./Build-Script/Make.sh
